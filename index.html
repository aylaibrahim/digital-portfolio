Capybara Game

Java JavaSwing MySQL University

This project was done in Apache NetBeans for Programming Technology class. The goal is to guide the capybara through the forest to gather baskets and avoid crocodiles. It features 10 randomly generated levels and 3 base player health. When the game ends, the player has the option to write their name and save their score in the database (JDBC), however only top 10 scores are displayed (based on speed & difficulty achieved within a round).

Reddit Sentiment Dashboard

Python MySQL Streamlit Vader Matplotlib API (praw) Personal

This project is a Reddit post + comment Sentiment Analysis Dashboard visualizer. It is written with python, and utilizes mutliple libraries in order to achieve the best result. With the Reddit API (specifically praw) and MySQL Workbench I created a connection to fetch and store a users data by username, specifically their post titels and comments with the links and numbers filtered out for more precise analysis. I used mathplotlib and wordcloud to display the information, and streamlit to run the app. Unfortunately it only works locally.
